# ika-bot

Python bot for playing web-based strategy game Ikariam

### Instructions

Put emails of your accounts to emails.txt

```python
python3 groups.py
```

Run strategy script:
```bash
./run_sstrategy.sh
```
Run transport script:
```bash
./run_transport.sh
```
